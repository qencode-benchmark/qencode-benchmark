# benchmarks/standard/

Contains the fixed QEncode Standard Benchmark Suite definitions.

- `suite_v1.yaml` - Standard Suite v1 (reference)
- `suite_v2.yaml` - Standard Suite v2 (current, active)

These YAML files define the canonical benchmark configurations:
molecules, basis sets, encodings, ansatz types, and execution settings.

**Do not modify these files.** They are the immutable benchmark standard.
Changes require a new suite version and corresponding leaderboard rules update.
