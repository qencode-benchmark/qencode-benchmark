# datasets/

Contains public leaderboard dataset releases and internal leaderboard CSVs.

## Structure

- `leaderboard/` - internal CSVs generated by the benchmark pipeline
- `qencode_leaderboard_v1/` - versioned public release snapshot

## Public release files

| File | Description |
|---|---|
| `leaderboard_accuracy.csv` | Best-accuracy rankings per molecule |
| `leaderboard_hardware_cost.csv` | Lowest hardware-cost rankings per molecule |
| `leaderboard_balanced.csv` | Best balanced-score rankings per molecule |
| `leaderboard_metadata.json` | Release metadata and provenance |

These files power the public leaderboard at
[qencode-benchmark.org/leaderboard](https://www.qencode-benchmark.org/leaderboard).
